# Email Delivery Agent

The final send step of an outreach campaign. Once recipients have been confirmed and drafts approved upstream, this agent hands the campaign to the email provider, watches the batch out the door, and reports back when every message is on its way.

## Works with

- Gmail

## Capabilities

- Send an approved outreach campaign through your connected email provider
- Track the send batch to completion and surface a final delivery summary
- Route every message through the workspace's configured send pipeline
